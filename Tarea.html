Hyper-text Transfer Protocol

Ejemplo de una dirección web con el protocolo HTTP

Definición

El protocolo de transferencia de hipertexto HTTP es un conjunto de reglas que permiten la transmisión de información através de enlaces de hipertexto en la web
En el sitio web de Wikipedia se nos define como el protocolo de comunicación que permite las transferencias de información a través de archivos (XML, HTML…) en la World Wide Web.
